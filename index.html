Memoization

-The process of storing already calculated values into an array or some data structure so you don't have to calculate them again.

This code works, but it gets really slow once n reaches the high 20s and beyond.

int stepPerms(int n)
{
if (n < 1)
return 0;
if (n < 3)
return n;
if (n == 3)
return 4;
return stepPerms(n - 1) + stepPerms(n - 2) + stepPerms(n - 3);
}
